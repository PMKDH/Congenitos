== Description ==
 
Fury Core add extra features to Fury WordPress theme, like theme metaboxes, import/export and more.
Plugin does not work with other themes than Fury WordPress theme released by Theme Vision.
https://theme-vision.com/fury-wordpress-theme
 
== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload `fury-core` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
